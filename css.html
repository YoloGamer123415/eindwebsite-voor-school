Geschiedenis

Håkon Wium Lie.

CSS werd voor het eerst voorgesteld door Håkon Wium Lie op 10 oktober 1994. [19] Op dat moment werkte Lie met Tim Berners-Lee op CERN. [20] Verscheidene andere stylesheet-talen voor het web werden rond dezelfde tijd voorgesteld, en discussies over openbare mailinglijsten en binnen het World Wide Web Consortium resulteerden in de introductie van de eerste W3C CSS-aanbeveling (CSS1) in 1996. In het bijzonder een voorstel van Bert Bos was invloedrijk; hij werd co-auteur van CSS1 en wordt beschouwd als mede-maker van CSS.

Style sheets bestaan ​​in de een of andere vorm sinds het begin van Standard Generalized Markup Language (SGML) in de jaren 1980, en CSS werd ontwikkeld om style sheets voor het web te bieden. Een vereiste voor een bladtaal voor een webstijl was dat stijlpagina's afkomstig waren van verschillende bronnen op internet. Daarom waren bestaande stylesheet-talen zoals DSSSL en FOSI niet geschikt. Met CSS kan de stijl van een document echter worden beïnvloed door meerdere stijlbladen in de vorm van 'cascading'-stijlen.

Naarmate HTML groeide, omvatte het een breder scala aan stilistische mogelijkheden om aan de eisen van webontwikkelaars te voldoen. Deze evolutie gaf de ontwerper meer controle over het uiterlijk van de site, ten koste van meer complexe HTML. Variaties in webbrowserimplementaties, zoals ViolaWWW en WorldWideWeb, maakten een consistent uiterlijk van de website moeilijk en gebruikers hadden minder controle over hoe webinhoud werd weergegeven. De browser / editor, ontwikkeld door Tim Berners-Lee, had stijlbladen die hard gecodeerd waren in het programma. De stijlpagina's kunnen daarom niet worden gekoppeld aan documenten op internet. Robert Cailliau, ook van CERN, wilde de structuur scheiden van de presentatie, zodat verschillende stijlbladen verschillende presentaties konden beschrijven voor afdrukken, schermgebaseerde presentaties en editors.

Verbetering van de webpresentatiemogelijkheden was een aandachtspunt voor velen in de webgemeenschap en negen verschillende stijlbladtalen werden voorgesteld op de mailinglijst in www-stijl. Van deze negen voorstellen waren er twee vooral van invloed op wat CSS werd: Cascading HTML Style Sheets en Stream-based Style Sheet Proposal (SSP). Twee browsers dienden als proeftuinen voor de eerste voorstellen; Lie werkte samen met Yves Lafon om CSS te implementeren in Dave Raggett's Arena-browser. Bert Bos implementeerde zijn eigen SSP-voorstel in de Argo-browser. Daarna werkten Lie en Bos samen om de CSS-standaard te ontwikkelen (de 'H' werd van de naam verwijderd omdat deze stijlbladen naast HTML ook op andere opmaaktalen konden worden toegepast).

Lie's voorstel werd gepresenteerd op de "Mosaic and the Web" -conferentie (later WWW2 genoemd) in Chicago, Illinois in 1994 en opnieuw met Bert Bos in 1995. Rond deze tijd werd het W3C al gevestigd en interesseerde zich in de ontwikkeling van CSS. Het organiseerde daarvoor een workshop onder leiding van Steven Pemberton. Dit heeft ertoe geleid dat W3C werk aan CSS toevoegt aan de deliverables van de HTML editorial review board (ERB). Lie en Bos waren de primaire technische staf op dit aspect van het project, met extra leden, waaronder Thomas Reardon van Microsoft, die ook deelnamen. In augustus 1996 presenteerde Netscape Communication Corporation een alternatieve stijlbladtaal met de naam JavaScript Style Sheets (JSSS). De specificatie is nooit afgemaakt en is verouderd. Tegen het einde van 1996 was CSS klaar om officieel te worden en de CSS level 1 Recommendation werd in december gepubliceerd.
